# EAIRA Project Layer

This README is the first Project Navigation artifact for EAIRA Project Layer documentation.

## Purpose

Provide a project-layer entry point for current project documents, milestones, templates, and specifications.

## Project Layer Structure

- `milestones/`: milestone-level project artifacts such as charters and progress reports.
- `templates/`: reusable Project Layer task templates and master task specifications.
- `specifications/`: Project Layer master specifications.
